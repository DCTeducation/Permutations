# Permutations

This repository teaches **permutations (order matters)** using:
- clear math explanations
- Python code examples
- exam-style practice problems

## What you’ll learn
- When a problem is a permutation
- The permutation formula P(n, r)
- How to verify answers using code

## Structure
- `lesson.md` – core teaching notes
- `exercises.md` – student practice
- `solutions.md` – worked solutions
- `code/` – Python demos
