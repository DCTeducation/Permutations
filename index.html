What is a combination?

Order does NOT matter. Changing the order does not create a new answer.
Example: {A, B, C} is the same group as {C, B, A}.

When do you use combinations?

  • Choosing a group / team / committee
  • Picking items where the order doesn‚Äôt change anything
  • Anytime you only care who is chosen, not the order

What do n and r mean?

n = total choices you have

Example: 12 students available ‚Üí n = 12

r = how many you choose

Example: choose 4 students ‚Üí r = 4

The combination formula

This removes all the different orders.
It counts each group only once.
Formula: C(n,r) = n! / ( r!(n‚àír)! )

C(5,2) = 10

If you can swap the order and nothing changes, it‚Äôs a combination.

Example 3: Not a combination

Not a combination. Those are different roles, so order matters.
